# 🗓️ 万年历主页

这是一个美观实用的万年历网页，可以显示公历、农历和节假日信息，支持年月切换。

## ✨ 功能特色

- 📅 **月历视图**：清晰展示每月日期排列
- 🎯 **日期高亮**：今天日期特殊标记
- 🌙 **农历显示**：显示农历日期和节气
- 🎉 **节假日**：自动标记法定节假日
- ⏮️⏭️ **快速切换**：支持月份和年份跳转
- 📱 **响应式设计**：手机、平板、电脑都好看
- 🎨 **现代界面**：渐变背景+毛玻璃效果

## 🚀 使用方法

### 方式1：本地打开
1. 下载所有文件到同一文件夹
2. 用浏览器打开 `index.html` 即可

### 方式2：部署到 GitHub Pages（公开主页）
1. 在 GitHub 创建仓库
2. 上传所有文件
3. 进入 Settings → Pages
4. Source 选择 "Deploy from a branch"
5. Branch 选择 `main` 分支，`/ (root)` 目录
6. 点击 Save，等待几分钟
7. 访问 `https://你的用户名.github.io/仓库名/`

## 📁 文件说明

- `index.html` - 万年历主页面
- `style.css` - 样式文件（可选）

## 💻 技术栈

- HTML5
- CSS3（Flexbox + Grid）
- JavaScript（原生）
- 农历算法库（lunar-javascript）

## 🎨 自定义

可以修改 `style.css` 中的颜色变量来改变主题：
- `--primary-color`：主色调
- `--bg-color`：背景色
- `--glass-bg`：毛玻璃背景色
